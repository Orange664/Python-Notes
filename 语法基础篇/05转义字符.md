**转义：转义是特殊意义，默认有特殊意义  \**

\\\ 反斜杠符号

\\' 单引号 或者 \\"

#### \n 输出时换行，换行符

```python
info = '你的名字是Lucy\n我的名字是wilia\n我们是好朋友'
print(info)
```

#### \t 排版，制表符

```python
strl = '网站\t\t\t域名\t\t\t\t\t\t\t\t\t年龄\t\t价值'

print('we\'re friend') #字符串内用引号需要转义
#print 默认自带换行符
```

#### print（‘输出的内容’，end = ‘\n’) 

end为结束符

```python
print(123,end = '') # 取消换行
print(234)
结果为 123234
#里面可以添加其他内容
print(123,end='*')
print(234)
结果为 123*234
```
