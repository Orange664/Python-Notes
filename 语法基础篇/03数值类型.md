######  在字符串内部使用双引号或单引号

```python
print("hello \"w\"orld")
```

输出的结果为 hello “w”orld

######  字符串的运算

"+" 字符串的拼接

```python
a = 'hello'
b = 'world'
print(a+' '+b)
结果为：hello world
```

"*"重复输出字符串

```python
a = 'hello'
print(a*2)
结果为：hellohello
```

in/not in	在/不在 里面

```python
print('h'in a)
结果为：True 表示字符串"h"在a里面
print('h' not in a)
结果为：False
```
